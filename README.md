# Hakoniwa AI

[![Demo](https://img.youtube.com/vi/19TpKIQXf9w/maxresdefault.jpg)](https://www.youtube.com/shorts/19TpKIQXf9w)

A palm-sized, fully local integrated machine learning laboratory designed for beginners and data science enthusiasts.

* **100% Offline:** Runs entirely on your smartphone. No external cloud APIs or data transmission required.
* **Kaggle-Style Data Science (NEW):** Import massive CSV datasets directly from your device. The app automatically handles complex preprocessing like missing value imputation and column filtering, allowing you to run batch predictions and generate submission files completely offline.
* **Multiple AI Architectures:** Experiment with Multilayer Perceptrons, Random Forests, and Variational Autoencoders (VAE) for image generation.
* **Miniature Language Model (SLM Mode):** Experience the fundamentals of Natural Language Processing (NLP). Build a lightweight model to predict the next character based on a 1-to-5 character context, demonstrating how modern LLMs understand sequences on a microscopic scale.
* **No Code Required:** Visually build, train, and analyze models using intuitive tools like Scatter Plots, Confusion Matrices, and Sensitivity Analysis.
* **Model Export (TinyML):** Export your trained inference logic as raw **C++, Rust, Python, or Dart** code. You can even run your models on 8-bit microcontrollers with ultra-low RAM.
* **Built for Mobile:** Developed with Flutter and Hive for a seamless, highly optimized, and memory-safe mobile experience.

## Development Note
This project was developed in collaboration with Generative AI. 

To be completely transparent, the author is not an AI expert or a data scientist. I am an absolute beginner who barely understands the complex mathematics behind AI and is just starting to learn the basic terminology. This app was born out of my personal desire to create a visual "sandbox" so I could finally understand how machine learning actually works. 

While the core concepts, UI/UX design, and app structure were directed by me, the complex mathematical logic and the majority of the source code were generated, refactored, and optimized through pair-programming with Generative AI models. 

I hope this "hand-made" laboratory helps fellow beginners who share the same curiosity and struggles in understanding AI!
